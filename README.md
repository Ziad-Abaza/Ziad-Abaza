## 💼 About Me 

مرحبًا بك في ملفي الشخصي! أنا **زياد حسن**، طالب في جامعة برج العرب التكنولوجية، ومطور ويب Full Stack متحمس للتكنولوجيا والبرمجة. بدأت رحلتي في عالم البرمجة من خلال حبي للحلول الإبداعية، وقد ساعدني هذا الشغف على بناء مشاريع متنوعة باستخدام تقنيات متعددة.

### 📌 ما أعمل عليه حاليًا:
- تطوير نموذج لعبة Tic-Tac-Toe باستخدام Deep Neural Networks.
- تحسين API في Laravel باستخدام Vue.js وSanctum.
- تعلم أساسيات الذكاء الاصطناعي: Linear Regression, Gradient Descent, DNNs...
- العمل على مشروع تخرجي: **مدينة ذكية متكاملة تعتمد على الذكاء الاصطناعي** لتحسين الاستدامة وجودة الحياة.
